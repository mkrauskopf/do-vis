Discrete Optimization Visualization
===================================

Deployed application might be tried [here](http://www.martyn.cz/do-viz/).

Installation
------------

* Install [node.js]
* `npm install -g grunt-cli`
* `npm install -g bower`
* `npm install`
* `bower install`

Running
-------

* `grunt server`
* go to [http://127.0.0.1:9000/](http://127.0.0.1:9000/)

[node.js]: http://nodejs.org/
